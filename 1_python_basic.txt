스크립트 언어 : 응용 프로그램 또는 소프트웨어 환경을 제어하는 프로그래밍 언어

computer hardware-->operating system-->application software

operating system 과 application software을 활용할 수 있게 도와주는 도구

활용분야 : 

시스템관리 및 자동화 -  unix shell script, perl, python, powershell

웹 개발 - javascript, php,ruby

게임 개발 - unity c# script, unreal engine blueprint

장단점:

장점 - 가성비 & 가심비 (쉽고, 개발이 빠름)

단점 - 느리고 특정 어플리케이션에만 활용될 수 있음
(인터프리터 방식,c나 c++의 경우 코드를 바이너리 코드로 변환 후 실행하지만,스크립트언어는 소스코드 그대로 실행)

추세:
-소프트웨어 개발 복잡도가 지속적으로 높아지고 있음

-개발을 빨리 해야함

-스크립트 언어의 활용폭이 높아지고 있음

-다양한 소프트웨어를 함께 사용하는 경우가 많아짐

대응 전략:
-전통적인 언어 c/c++에 대한 이해

-파이썬 필수 - 글루 언어(glue language), ai 언어

-응용분야별 필요에 따른 핵심 스크립트 언어 숙지(모바일/웹 java, unreal blueprint, mac os swift)
